setOrUpdateInArray​

▸ setOrUpdateInArray<T>(array, index, producer): void

Sets or updates the given key in the given array.

The producer can be a function, that will only receive a value, if it's present. This means that arguments.length will be 0, if the key doesn't exist yet. You may pass in a function, if you can easily handle undefined in an unambiguous way, e.g. setOrUpdate(array, key, (count) => (count ?? 0) + 1) .

The producer may also be an object with onSet and onUpdate, which provide stronger typings than the single function for both cases.
